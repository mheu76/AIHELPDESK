## Screenshot Guidelines

- **Resolution**: 1400x800px or higher (maintain 16:9 aspect ratio)
- **Format**: PNG (preferred) or JPG
- **Quality**: High quality, not compressed
- **Content**: 
  - Use realistic but sanitized data (no real user info)
  - Show key features clearly
  - Light mode screenshots preferred
  - Korean UI preferred (matches project locale)

## How to Capture

1. Run the application in development mode
2. Log in as different user roles (employee, IT staff, admin)
3. Use browser dev tools to set viewport to 1400x800
4. Capture screenshots with browser screenshot tools or Snipping Tool
5. Save to this directory with exact filenames above
